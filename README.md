0_data_generation.R: codes for simulation data generation

1_PLR_functions.R: codes for all functions of PLR model, used in both simulation and application

2_PLR_simulation.R: codes to run simulations of PLR model for multiple treatments

3_IRM_simulation.R: codes to run simulations of IRM model for multi-valued regimens

4_HIV_application.R: codes to run HIV-kidney disease application

Note the simulation is designed to run in a multi-core cluster and it takes a relatively long time. Most of those codes were wrote during late 2024-early 2025 with relatively less support of AIs comparing to today's AI scenes.
